# AI的本质

### 官方解释

> The theory and development of computer systems able to perform tasks normally requiring human intelligence. ——— Oxford Dictionary

能够执行通常需要人类智能才能完成的任务的计算机系统的理论与发展。

### 通俗解释

> Using data to solve problems.

让机器从数据中找到规律，从而解决实际问题。就是AI利用数据，解决问题。

数据 -> 机器学习model -> 问题。

### 解决问题过程

Data(数据) -> Training(训练) -> Model(大模型) -> Prediction(预测) -> solve problems(解决问题)

**训练阶段：** 通过对数据的训练，创建一个预测模型并对其进行微调。

**模型生成：** 预测模型可以从这些数据背后找出答案来，帮我们解决某个问题。

**预测阶段：** 通过测试集完成模型评估，从而了解模型在测试集中的有效性。

**不断改进：** 过程中，预测模型会被不断改进和使用。

# AI机器学习

### 机器学习的七步骤

收集数据 -> 准备数据（数据预处理） -> 选择模型 -> 训练 -> 评估 -> 超参数调整 -> 预测

### 机器学习的训练过程

在机器学习中，有很多特征对结果有影响，基于这些特征我们需要训练Model中的权重（w），这些特征值构成的矩阵称之为权重矩阵（weights）。

同时，还会存在偏差，称之为变量（biases）。

机器学习的过程就是在搜索空间中对W和B进行搜索的过程，使得模型的准确率达到某个标准。

一个训练步骤（training step），称之为一次迭代，目标在于更新权重（W）和变量（b）。

**一次迭代：** Training Data -> Test & Update W,b -> Prediction -> Mode(W,b)

通过多次迭代，模型中的参数不断进行更新，就好像在数据中进行线性拟合。

### 机器学习的模型选择

##### 1、回归问题和分类问题

**回归问题：** 结果是连续值，是一个趋势。比如：房价预测，这个问题的结果是一个趋势，这一类的问题就是回归问题。

**分类问题：** 结果是有限个数。比如：液体是水还是饮料，这个问题的结果是有限的，这一类的问题就是分类问题。

##### 2、如何判断一个问题是分类问题还是回归问题

我们可以看这个问题输出结果的数据类型是什么，如果是离散的则是分类问题，如果是连续的则是回归问题。

##### 3、线性回归和逻辑回归

**线性回归：** 核心思想是预测一个连续的数值，可以把线性回归理解成“找趋势线”。它试图找到一条最佳拟合的直线，来描述变量（X）和因变量（Y）之间的线性关系。

**逻辑回归：** 预测一个事件的概率，通常用于分类。虽然名字有“回归”，实际上就是来解决分类问题的。它的核心不是为了拟合一条直线，而是判断是什么分类或者预测出一个概率值（通常在0-1之间）。

| 特征       | 线性回归               | 逻辑回归                           |
| -------- | ------------------ | ------------------------------ |
| **任务类型** | **回归**             | **分类**（主要是二分类）                 |
| **输出结果** | **连续的数值**（可以是任意实数） | **离散的类别**，或属于该类别的**概率**（0到1之间） |
| **核心函数** | 线性函数 (Y=βX+c)      | Sigmoid函数 (S(z)=1/(1+e−z))     |
| **预测内容** | 具体数值（如：价格、温度）      | 类别标签（如：是/否、猫/狗）                |
| **关系拟合** | 用直线/平面拟合数据点        | 用S形曲线寻找决策边界                    |
| **损失函数** | 均方误差 (MSE)         | 交叉熵损失 (Log Loss)               |

##### 4、需要多少数据，机器才能学习到规律

**简单问题：** 模型不复杂，所需要的数据量可以比较少（几十个）

**复杂问题：** 模型复杂，需要数据量比较大（几千到几万不等，具体和受影响的特征数量成正比）

### 机器学习的特征构造

##### 1、如何用线性回归模型拟合非线性关系

**欠拟合：** 预测出的结果和实际结果偏差比较大

**过拟合：** 虽然预测值和结果值偏差不大，但并不是一个正态分布。

**拟合：** 预测值和结果值偏差不大，同时也是一个正态分布。

奥卡姆剃刀定理 => 越简单越有效；

### 机器学习的评估

通常选择一部分数据去做训练，需要留10%-30%去做预测。数据量少我们尽可能留10%做预测。如果没有预留预测数据就不知道我们训练出来的模型是否有效。

**有效：** 至少预测正确率要80%，才有效，否则还需要添加更多的数据训练或超参数调整（一般正确率离80%比较近，比如78%可以通过调参）。

### 超参数调整（调参）

**超参数：** 能在Model外围可进行人工调整的参数进，比如epoch次数、学习率等。比如洗衣机人工可操作洗衣时长、甩干力度等。

超参数调整是一个实验的过程，需要实际去调，然后对比。

通常在用AI的时候，不建议一开始就进行超参数的调整，应该多思考，有哪些关键特征，会影响到模型的预测结果。

# 数据分析模型

### 十大分析模型

**分类算法：** C4.5，朴素贝叶斯（Naive Bayes），SVM，KNN，Adaboost，CART

**聚类算法：** K-Means，EM

**关联分析：** Apriori

**连接分析：** PageRank

### 四大分类理解

##### 1、分类

针对已有的概念对数据进行分类，比如说性格，是I人还是E人，通过特征来最终做一个分类

##### 2、聚类

聚类是一种降维思维，对于大数据量，没有办法进行人工分类，让机器协助我们对数据按照特征进行划分。比如对于50000条考勤记录，划分出早鸟、爱迟到、踩点侠进行数据划分。

##### 3、关联分析

通过数据的关联性来分析，比如A和B经常被一起提起，说明A和B是有关联的。

##### 4、连接分析

分析连接，比如50000个人中，找到最多人认识的人，那个人就是有影响力的人。
